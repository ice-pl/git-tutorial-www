Tworzenie i obsługa gałęzi
  1. pobieramy projekt z repozytorium zdalnego
    git pull origin master
  2. tworzymy nową gałąź
    git branch feature
  3. przechodzimy na gałąź feature
    git checkout feature
  4. tworzymy nowy plik
    echo >> feature.txt
  5. sprawdzamy bieżący status
    git status
  6. dodajemy pliki do kolejki
    git add .
  7. tworzymy commit
    git commit -m "Dodanie pliku feature.txt"
    widok katalogu gdy jesteśmy na gałęzi feature
  8. przechodzimy na gałąź master
    git checkout master
    widok katalogu gdy jesteśmy na gałęzi master
git pull origin master git branch feature git checkout feature git status git add . git commit -m "Dodanie pliku feature.txt"
Łączenie gałęzi
  1. łączymy gałąź feature z master
    git merge feature
    widok katalogu po dołączeniu gałęzi feature do master
  2. usuwamy gałąź feature feature z projektu
    git branch -d feature
git merge feature git branch -d feature
