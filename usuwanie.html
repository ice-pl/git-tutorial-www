Usuwanie commit-ów
  1. tworzenie dodatkowych commitów
    echo > pierwszy.txt
    git add .
    git commit -m "pierwszy commit testowy"
    git push
  2. ustawienie ostaniego commita do zachowania
    git reset --hard _nr_commita_do zachowania_
  3. nadpisanie danych na serwerze
    git push --force
git add . git commit -m "pierwszy commit testowy" git push git reset --hard _nr_commita_do zachowania_ git push --force
Usuwanie gałęzi
  1. tworzenie dodatkowych gałęzi
  2. usuwanie brancha lokalnego
    git branch
    git checkout master
    git branch -D second
    git push
  3. usuwanie brancha zdalnego
    git push https://github.com/_nazwa_użytkownika_/learn-git.git --delete second
git branch git checkout master git branch -D second git push git push https://github.com/_nazwa_użytkownika_/learn-git.git --delete second
