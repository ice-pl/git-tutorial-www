Tworzenie nowego repozytorium na githubie
  1. tworzymy nowy projekt
  2. nadajemy nazwę
  3. kopiujemy ścieżkę https projektu
Obsługa repozytorium na githubie
  1. podłączamy repozytorium lokalne do repozytorium zdalnego
    git remote add origin https://github.com/_nazwa_użytkownika_/learn-git.git
  2. wgrywamy lokalne repozytorium na serwer
    git push origin master
  3. sprawdzamy na stronie co się stało po stronie serwera
  4. pobieramy wersję z serwera
    git pull origin master
    w tym przypadku nic się nie pobrało bo lokalna i zdalna wersja były takie same
git remote add origin https://github.com/_nazwa_użytkownika_/learn-git.git git push origin master git pull origin master
Pobieranie repozytorium z githuba
  1. pobieramy adres ze strony githuba
  2. tworzymy nowy folder i wchodzimy do niego
  3. pobieramy wersję z serwera do nowego katalogu
    git clone https://github.com/_nazwa_użytkownika_/learn-git.git
